# Applied-computer-vision
This repository contains a collection of computer vision experiments performed as part of practical exploration in image processing and deep learning.

 overview

The notebook includes hands-on experiments covering image preprocessing, feature extraction, model development, and evaluation techniques. The objective of this project is to build strong foundational and applied knowledge in computer vision.

---

##  Experiments Included

- Image loading and visualization
- Image preprocessing and normalization
- Data augmentation techniques
- Convolutional Neural Networks (CNN)
- Model training and validation
- Performance evaluation (Accuracy, Confusion Matrix)
- Error analysis and observations

---

##  Technologies Used

- Python  
- NumPy  
- OpenCV  
- Matplotlib  
- Scikit-learn  
- TensorFlow / Keras  

---

##  Results

The experiments demonstrate how preprocessing techniques, augmentation strategies, and model architecture choices influence performance. Detailed outputs, plots, and model evaluation metrics are included inside the notebook.
